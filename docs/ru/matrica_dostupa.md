
## Матрица доступа Cloud Backup

| **Действие** | **Права доступа** | **Видимость в UI** |
| --- | --- | --- |
| **Раздел «Планы резервного копирования»** | `owner/member/reader` | Отображается |
| Просмотр списка планов | `owner/member/reader` | Список отображается |
| Создание плана | `owner/member` | Кнопка отображается |
| Остановить/Включить план | `owner/member` | Кнопка отображается |
| Редактирование плана | `owner/member` | Кнопка отображается |
| Удаление плана | `owner/member` | Кнопка отображается |
| **Раздел «Резервные копии»** | `owner/member/reader` | Отображается |
| Просмотр списка резервных копий | `owner/member/reader` | Список отображается |
| Восстановление диска из копии | `owner/member` | Кнопка отображается |
| Удаление резервной копии | `owner/member` | Кнопка отображается |
