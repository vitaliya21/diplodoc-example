
## Список проектов

После регистрации в [FKK Cloud](https://cloud.fkk.ru "cloud.fkk") у пользователя автоматически создается дефолтный проект с ролью owner. Чтобы просмотреть идентификатор проекта (Fkk[10 цифр]) и их список используйте выпадающее меню:

![alt text](image-56.png)

Чтобы перейти на другой проект, нажмите на него мышкой. 

Каждому проекту соответствует его баланс проекта, на который можно переводить средства для оплаты услуг. [Подробнее](perevod.md)
