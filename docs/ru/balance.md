
## Баланс

После регистрации в [FKK Cloud](https://cloud.fkk.ru "cloud.fkk") у пользователя автоматически создается: основной баланс, баланс проекта, бонусный баланс. Чтобы просмотреть список балансов используйте выпадающее меню:

![alt text](image-59.png)

{% note info %}

1. Основной баланс — средства, которые пополняются с [банковской карты](by-card-with.md) или [банковским переводом](bank-perevod.md).

2. Бонусный баланс — средства, начисляемые пользователю бесплатно в виде бонуса.

3. Баланс проекта — средства, которые пополняются исключительно переводами с основного баланса.

{% endnote %}
